# vue_memo_app

## 참고서적 : 커피 한 잔 마시며 끝내는 Vue.JS

### CODE 5-1
    - vue init webpack-simple memo-application

    - 책에서 사용한 옵션
        -Project name: memo-application (자유롭게 작성)
        -Project description: A Vue.js project (자유롭게 작성)
        -License: MIT
        -Use Sass?: NO
    - 애플리케이션 실행에 필요한 모듈 설치
        -cd memo-applicaion
        -npm install    혹은    npm i

    - 프로젝트 실행
        -npm run dev
